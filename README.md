# Slack-chat
### Simple analog of Slack chat for text messaging, adding, editing and removing channels.
This project was built on Hexlet courses for practicing:

- React
- Redux + Redux Forms
- Websockets
- Ajax, REST
- Bootsrap on React
- Deploy (heroku.com)
